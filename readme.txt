Описание
--------
Консольная реализация игры "Реверси", с возможностью установки произвольного размера поля и игры с Искуственным Интеллектом (далее ИИ).

Справка по использованию
--------
Приложение доступно только для запуска через консоль.
Игра доступна в двух режимах: офлайн или онлайн.

1. Офлайн.
В офлайне игра может идти в двух режимах: человек-человек или человек-ИИ. Рассмотрим каждый режим по отдельности.
ЧЕЛОВЕК-ЧЕЛОВЕК
В таком режиме игра ведется на одном компьютере, но двумя людьми (можно, конечно, играть и одному, но это как минимум странно). Для запуска приложения достаточно написать следующую команду:
	reversi.py
Возможно дополнительно указать размер поля для игры в формате "ШИРИНАxВЫСОТА" с помощью команды "--size" ("-s"). Если этот параметр не указывать, то игра будет вестись в режиме 8х8. Минимальные размеры поля ограничены занчением 2x2 НЕвключительно; максимальные размеры поля не ограничены, однако при размерах, больших 78x13, возможны проблемы с прорисовкой поля из-за ограничений на размеры окна консоли.
***Пример запуска игры с разными размерами полей:
	reversi.py --size 10x23
	reversi.py -s 6x9
	reversi.py

Также присутствует возможность играть с так называемыми "камнями" с помощью параметра "--rocks" ("-r"): на поле будут в случайный местах будут расположены несдвигаемые фишки специального типа, на место которых будет невозможно сделать ход ни одному из игроков.
Кол-во камней, добавленных в игру, будет равно кол-ву раз, сколько указан параметр. Если же данный параметр не указан, камни в игру не добавляются.
***Примеры запуска игры с "камнями":
	reversi.py --rocks                //будет добавлен один камень
	reversi.py --rocks --rocks        //будет добавлено два камня
	reversi.py -rrrr                  //будет добавлено четыре камня

ЧЕЛОВЕК-ИИ
Чтобы сыграть с компьютером, необходимо и достаточно при запуске указать параметр "--ai" или "-a".
Присутствует возможность указать, за какую сторону будет играть ИИ - "--black" или "--white" ("-b" или "-w").
Также остается возможность указывать размеры игрового поля с помощью параметра "--size" и играть с камнями с помощью параметра "--rocks".
***Примеры запуска приложения с ИИ:
	reversi.py --ai
	reversi.py -ai -black
	reversi.py -s 49x2 -a
	reversi.py -s 9x11 -a -w
Важное примечание: невозможно запустить приложение, выбрав сторону, за которую будет играть ИИ, но при этом не указав параметр "--ai" ("-a").


2.Онлайн
В данном режиме можно играть с другим человеком, находящимся за другим компьютером.
Для игры по сети предусмотрены две команды: команда для создания новой игры и команда для присоединения к уже созданной игре.
Команда для создания новой игры: --newgame НОМЕР_ПОРТА (-n НОМЕР_ПОРТА)
При создании новой игры возможно также использовать команды для указания параметров игры (которые описаны в режиме игры офлайн.человек-человек). Играть по сети с ИИ нельзя.
***Примеры запуска приложения с созданием новой игры:
	reversi.py --newgame 80
	reversi.py -n1 -s 10x10
	reversi.py -n100 -s 12x4 -rrr
После запуска приложения Вы автоматически подключитесь к серверу и Вам останется только дождаться подключения соперника.

Команда для присоединения к уже созданной игре.
Для подключения к уже существующей, но еще не начатой игре Вам необходимо знать IPv4-адрес компьютера, создавшего эту игру, и номер порта, по которому эта игра будет идти. При выполнении этих условий, подключиться к созданной игре можно с помощью команды --connect АДРЕС:ПОРТ (-c АДРЕС:ПОРТ).
***Примеры подключения к существующей игре:
	reversi.py --connect 192.168.1.5:1
	reversi.py -c 169.54.212.31:123
Пирмечание: теоретически, при подключении к существующей игре Вы можете указать параметры игры - вроде размера или кол-ва камней, - но смысла в этом нет, так как игра уже создана с теми параметрами, которые указал создатель игры; параметры, указанные Вами будут просто проигнорированы.



Пример игровой ситуации
--------

"""
Белые: 6
Черные: 7
Текущий игрок: Белые (○)
  01234567
0 -------#
1 .-..----
2 -●-●--#-
3 -.●○○○..
4 ---●●●●-
5 --.○○○..
6 --------
7 -----#--
  01234567
Ваш ход:
"""

Символ "#" обозначает камни - на их места нельзя сходить. Символом "." обозначены доступные в текущий момент времени позиции для ходов. Чтобы сделать ход, нужно выбрать один из доступных ходов и отправить его в следующем формате: ГОРИЗОНТАЛЬНАЯ_КООРДИНАТА РАЗДЕЛИТЕЛЬ ВЕРТИКАЛЬНАЯ_КООРДИНАТА. Под "разделителем" имеется в виду любые нечисловые символы. Отсчет координат ведется с нуля.
***Примеры коррекной записи ходов:
	3, 2
	5;4
	8 0
	15 - 9

В случае, если ход сделан неверно, на консоль выведется соответсвующее сообщение.
Если игрок, который должен ходить следующим, не имеет доступных ходов, ход переходит к другому игроку. Если доступных ходов нет ни у одного из игроков, игра заканчивается.
